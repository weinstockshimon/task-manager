# Task Manager

Mobile-first task manager for Awning Elegance.

## Features
- Master categories
- Categories/jobs
- Tasks and subtasks
- Calendar view
- Archive
- Search
- Undo actions
- Inline editing
- Local save support
- Mobile-focused UI
